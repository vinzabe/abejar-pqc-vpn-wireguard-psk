# Abejar PQC Library - Kyber-1024

This directory contains the encrypted post-quantum cryptography implementation.

## Files

| File | Description |
|------|-------------|
| `kyber-keygen.bin` | Key generation binary (encrypted) |
| `kyber-encaps.bin` | Key encapsulation binary (encrypted) |
| `kyber-decaps.bin` | Key decapsulation binary (encrypted) |
| `libkyber.so.enc` | Shared library (encrypted) |
