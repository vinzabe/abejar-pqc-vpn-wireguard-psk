# Routing Configuration

This directory contains the encrypted routing logic for the PQC VPN Router.

## Files

| File | Description |
|------|-------------|
| `setup-routes.sh.enc` | Network routing setup (encrypted) |

## Functionality

The routing script handles:
- Docker network preservation
- VPN tunnel routing
- Split tunneling (optional)
- Failover configuration
